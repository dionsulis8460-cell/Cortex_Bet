## 🤖 Treinar IAs (Optuna - Recomendado)

**Sistema possui 2 IAs:**

1. **Analyst AI (ProfessionalPredictor)** - Ensemble Multi-Modelo
   - Usado no CLI opção 7 (Scanner) para gerar predições base
   - **Algoritmos:** **LightGBM + XGBoost + CatBoost** (Stacking Ensemble)
   - Distribuições: Tweedie (LightGBM), Poisson (XGBoost, CatBoost)
   - Meta-Learner: Ridge Regression com votação ponderada automática
   - Acurácia atual: 65% → Meta: 68-69% (pós-ensemble v3.0)

2. **Manager AI (SelectionStrategy)** - Ranqueamento inteligente
   - Usado no Scanner para selecionar Top 7 melhores picks do dia
   - Critérios: Probabilidade calibrada + Fair Odds + Edge
   - Acurácia atual: 100% (7/7)

**Comandos de Treinamento (Completos e Robustos):**

Para manter **100% das inteligências do Cortex Bet** perfeitamente calibradas e atualizadas para a Manager AI (Scanner), você deve executar ambos os comandos abaixo em sequência:

**1º Passo: Treinar Motor Principal e Calibrador (Analyst AI via Ensemble V3.0):**
```powershell
python scripts/train_model.py --experiment "Cortex_V5_Production" --optuna --trials 100
```
*Nota: Aumente trials para 100+ em produção real para maximizar a otimização de hiperparâmetros.*

**2º Passo: Treinar a Rede Neural Auxiliar (Para apoiar a Manager AI / Safety Over):**
```powershell
python src/ml/train_neural.py --experiment "Cortex_V5_Neural_Prod" --trials 100
```

**Configuração Atual:**
- Mode: Full Production Pipeline
- Modelos: LightGBM + XGBoost + CatBoost (Stacking)
- Otimização: Optuna (Hyperparameters)
- Calibração: Automática (Isotonic/Sigmoid)

- 🤖 **IA Avançada (Cortex V2.1 Calibrated):** 1 predição principal
- Exemplo: "Under 10.5 Corners"
- Usa Temperature Scaling para calibração probabilística
